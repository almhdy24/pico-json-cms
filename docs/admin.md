# Admin Panel

## Login
- URL: `/admin/login`
- Default: `admin / secret123`
- **Change immediately!**

## Dashboard
- Add New Post
- All Posts (edit/delete)
- Settings
- Logout

## Posts Management
- Add, Edit, Delete
- Slug auto-generated
- Content supports HTML

## Security Features
- Passwords hashed
- Session regeneration
- Limited login attempts
