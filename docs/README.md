# pico-json-cms Docs

**A simple flat-file CMS that stores everything in JSON files. No database needed.**

> ⚠️ **Note:** This is **pico-json-cms**, not related to Pico CMS. They're different projects.

**Live demo:** [pico-json-cms.alwaysdata.net](https://pico-json-cms.alwaysdata.net)

## What is this?

pico-json-cms is a PHP-based content management system that:
- Stores all data in `content/` JSON files (not `data/`)
- Has a simple admin panel for managing content
- Is lightweight and developer-friendly
- Can be extended with plugins

## Quick Start

1. Download the files
2. Upload to any PHP web server
3. Visit `/admin/login`
4. Default login: `admin` / `secret123`
5. Start writing posts

## Why use this?

| Good for... | Not so good for... |
|-------------|-------------------|
| Personal blogs | Large e-commerce sites |
| Documentation sites | Multi-user platforms |
| Simple business sites | Complex web apps |
| Learning PHP/CMS | High-traffic sites |

## Core Features

- **Posts management** - Create, edit, delete blog posts
- **JSON storage** - All data in readable JSON files
- **Admin panel** - Password-protected dashboard
- **Theme system** - Easy to customize appearance
- **Plugin hooks** - Add custom functionality
