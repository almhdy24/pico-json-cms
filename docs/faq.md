# FAQ

## General
- **Is this Pico CMS?** No, it's pico-json-cms.
- **Database needed?** No, uses `content/` JSON files.
- **Free?** Yes, open-source.

## Troubleshooting
- White page: check PHP version 7+
- Posts missing: verify `content/posts.json`
- Can't save: check folder permissions
- Admin login issues: clear cookies, check password hash

## Customization
- Images: create `uploads/` folder
- Markdown: use plugin
- Contact form: use plugin

## Performance
- Best for small to medium sites
- 500 posts max comfortably
