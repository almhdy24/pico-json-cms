# Pico JSON CMS

A lightweight PHP CMS using JSON for storage, Markdown support, and a simple admin panel.

## Features
- Markdown posts
- Tiny admin panel
- Site settings (title, description, keywords)
- Pagination support
- Flash messages

## Requirements
- PHP 8+
- Web server (Nginx/Apache)

## Installation
1. Clone the repository
2. Configure your web server
3. Start adding posts via admin panel
